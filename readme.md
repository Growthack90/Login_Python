# LOGIN PYTHON

## Requirement:
- Install Python3
- Install Virtual Environment (VENV)
    * py -3 -m venv .venv
    * .venv\Scripts\activate
    * deactivate    # to deactivate develop environment
- Install PIP
- Install dependencies:
    * pip install setuptools
    * pip install flask
    * pip install flask-login

- Check if installed correctly:
    * pip list

- Run App
    * flask --app app run
